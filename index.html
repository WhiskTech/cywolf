Cywolf

  

evented, modular wolfgame for node

Install on NPM Play on IRC View on GitHub

Cywolf is an implementation of the popular party game, Mafia, in node.js. It was originally coded to be used in the Cyril IRC bot, but was recoded and made modular.

Features

  • Modularized - use it anywhere
  • Extensible - easy Roles API adds support for multiple custom roles
  • Simple - new Wolfgame(); and you're off!

Docs

Since this is beta, there is no final documentation yet. You can have a look at bot2.js from the cyril project to see how this is used in an IRC-type context.
